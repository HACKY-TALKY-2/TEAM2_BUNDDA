# 번호따기 Project!
주변에 관심이 가는 사람이 있으신가요? 번따 웹에 접속해서 프로필을 확인하고 대화해보세요!

## 기능

### 1. 만남의 광장
-만남의 광장에서 접속한 사람들의 프로필 확인
-관심있는 사람에게 대화 요청

### 2. 프로필 입력
-대화 전에 만남의 광장에 입장하기 위해서 프로필 입력
-닉네임, 나이, 전화번호, 프로필 사진 입력
-닉네임 중복 확인

### 3. 채팅
- 3분안에 당신과 잘 맞는지 확인해보세요
- 간단한 미니게임과 대화를 할 수 있습니다.
- 3분 뒤 모든 대화 내용과 대화방은 폭파됩니다.
